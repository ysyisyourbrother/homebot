# 声纹检测

声纹检测用于识别当前说话的家庭成员。启用后，homebot 会将说话声音与已登记的声纹进行比对。

## 启用声纹检测

执行以下命令：

```bash
python -m homebot config
```

选择 **[3] Voice Channel**，再进入 **[4] User Recognition Settings**，选择启用用户识别。

首次启用时，系统会自动下载声纹检测模型，并保存到工作区语音目录的 `model/` 下：

```text
~/.homebot/workspace/voice/model/
```

## 管理家庭成员

在 **User Recognition Settings** 中选择管理用户档案，即可添加、重新录制或删除家庭成员。

添加成员时，需要按提示录制几段自然朗读的语音。系统会将录音处理为该成员的声纹特征向量。后续有人说话时，homebot 会把当前声音与已登记的特征向量进行匹配；匹配结果超过识别阈值时，即可识别出对应成员。

建议使用日常交互时的麦克风，并在相近的环境中完成录制，以获得更稳定的识别效果。

## 用户模板

添加家庭成员后，工作区会为该成员创建个人档案：

```text
~/.homebot/workspace/members/<成员标识>/USER.md
```

这份文件不再是访客模板，而是该成员的用户模板。记忆模块会持续维护其中的姓名、偏好等用户信息，使 homebot 能根据识别到的家庭成员提供更贴合的交互。
