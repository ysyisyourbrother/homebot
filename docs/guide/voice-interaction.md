# 语音唤醒与对话

启动网关并启用语音通道后，homebot 会使用 KWS（关键词唤醒）模型持续监听你在初始化时设置的唤醒词。说出该唤醒词后，系统会播放回应声，例如“嗨”；回应声结束后即可开始说出你的问题。

::: tip 提示
等待回应声结束再开始提问，能避免语音识别遗漏开头内容。
:::

如果唤醒后 10 秒内没有说话，homebot 会自动退出本次对话。如果暂时没有问题，也可以直接说“再见”或“退出”，立即结束对话。

## 对话方式

### 单轮对话

适合信息完整、可以直接回答的问题。你说完后，homebot 会给出回复，本次对话随即结束。

例如：“今天天气怎么样？”

### 追问补充

当你的需求缺少必要信息时，homebot 会主动追问。补充完所需信息后，它会完成任务并结束对话。

例如，你说“帮我设置一个闹钟”，但没有说明时间和提醒内容；homebot 会询问具体几点、提醒什么事。

### 多轮对话

如果你明确希望深入讨论某个主题，homebot 会保持连续交互。

例如：“和我聊聊什么是神经网络。”

多轮对话会持续进行，直到你说出“退出”或“再见”结束对话。
