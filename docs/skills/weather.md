# 天气查询

Weather Skill 通过 [wttr.in](https://wttr.in/) 获取天气信息，无需 API Key，开箱即用。

## 功能

- 查询当前天气（温度、湿度、风力、天气状况）
- 查询多日天气预报
- 支持中文城市名

## 前置依赖

- `curl` — 系统自带

## 使用示例

> "今天天气怎么样"
> "北京明天什么天气"
> "上海这个周末的天气"
> "深圳未来三天天气预报"

默认城市为 **广州**（未指定城市时）。

## 工作原理

Weather Skill 通过简单的 `curl` 命令调用 wttr.in：

```bash
# 当前天气
curl -s "wttr.in/<城市>?format=%l:+%c+%t+%h+%w&m"

# 三日预报
curl -s "wttr.in/<城市>?T&m&2" | head -40
```

中文城市名直接使用，英文城市名中的空格用 `+` 编码。

## 不需要 API Key

wttr.in 是免费服务，无需注册。但请注意：
- 请求频率有限制
- 它是一个公共服务，不适合商业/高并发场景
